# Gestión de Tareas

Aplicación web para gestionar tareas con autenticación de usuarios.

## Instrucciones

1. Instalar dependencias: `npm install`
2. Ejecutar: `npm run dev`
3. Abrir en navegador: http://localhost:5173

## Despliegue

Despliegue automático con GitHub Actions al hacer push a master.

## Funcionalidades

- Registro e inicio de sesión
- Crear, editar, eliminar tareas
- Marcar tareas como completadas
- Buscar y filtrar tareas
- Datos almacenados en LocalStorage

## Framework

React con Tailwind CSS.
